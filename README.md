# CV_report2019

慶應義塾大学 理工学研究科 「Computer Vision特論 (2019春)」の期末レポート．

### Description

レポートは`docs/main.pdf`に，実装に使ったコードは`code`にある．

### Requirements

- OS 
    - macOS Mojave 10.14.4
- App
    - QuickTime Player 10.5 (935.3), iMovie (10.1.12)
- Language
    - Python 3.7.4
        - Numpy 1.16.1
        - Opencv-python 4.1.0

### Usage

```
$ python3 run_chaplin.py

$ python3 run_car1.py

$ python3 run_car2.py

$ python3 run_bus1.py

$ python3 run_bus2.py
```
